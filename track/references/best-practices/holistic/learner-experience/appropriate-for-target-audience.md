# Appropriate for Target Audience

The track should match its stated skill level in terminology, explanations, and challenge complexity -- evaluated across all challenges. The target audience is defined in the track plan (audience section).

## Criteria

| Score | Level | Description |
|-------|-------|-------------|
| 1 | Poor | The content does not match the stated skill level. A beginner track uses unexplained jargon, or an advanced track over-explains basic concepts. The mismatch is pervasive. |
| 2 | Below Standard | The content mostly matches the stated level but has multiple sections that are too advanced or too basic for the target audience. |
| 3 | Adequate | The content generally matches the target audience. One or two sections are slightly misaligned (e.g., a beginner track that briefly assumes intermediate knowledge). |
| 4 | Good | The content consistently matches the stated skill level. Explanations, terminology, and challenge complexity are appropriate. One minor audience mismatch exists. |
| 5 | Excellent | Every section is precisely calibrated to the target audience. Beginner tracks explain all concepts, use analogies, and provide step-by-step guidance. Advanced tracks respect the learner's expertise. No mismatches. |

**Beginner tracks:**
- Explain every new concept before using it
- Don't assume familiarity with tools or terminology
- Provide explicit, step-by-step instructions
- Use analogies and real-world comparisons

**Intermediate tracks:**
- Can reference common concepts without full explanation
- Expect users to extrapolate from examples
- Provide guidance but not every keystroke

**Advanced tracks:**
- Can use technical terminology freely
- Focus on problem-solving, not step-following
- Minimal hand-holding

## What to Watch For

- Inconsistent difficulty level (suddenly much harder without progression)
- Content that talks AT users rather than guiding them
- Beginner tracks that over-explain to the point of being patronizing — explain what's needed, not everything possible
- Advanced tracks that under-explain novel concepts just because the audience is advanced (new concepts still need introduction regardless of audience level)
