# Cross-References

When content references earlier material, it should do so consistently and without contradicting prior explanations.

This applies across all content types: assignment.md instructions, assignment steps, check script feedback messages, solve scripts, notes/tabs, and quizzes.

When content references earlier material:
- Use the same terminology
- Don't contradict earlier explanations
- Don't re-explain at length (trust prior learning)

## What to Watch For

- Contradictions between earlier and later explanations of the same concept
- A concept explained fully in challenge 1 being re-explained at length in challenge 5
- A check script failure message that uses different terminology than what the assignment taught
- Assignment steps or feedback messages that contradict how a concept was originally explained
