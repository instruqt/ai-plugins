# Managed Kubernetes Reference Index

Managed Kubernetes patterns combine sandbox compute with cloud-managed Kubernetes services. Each cloud provider's managed Kubernetes has different provisioning, auth, and networking.

- `container-plus-eks.md` — cloud-client provisions or interacts with AWS EKS
- `container-plus-aks.md` — cloud-client provisions or interacts with Azure AKS (and optionally ACR)
- `container-plus-gke.md` — cloud-client provisions or interacts with GCP GKE
- `vm-kind-plus-eks.md` — VM with local Kind cluster paired with a remote EKS cluster
- `vm-kind-plus-aks.md` — VM with local Kind cluster paired with a remote AKS cluster
