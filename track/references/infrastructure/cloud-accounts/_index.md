# Cloud Accounts Reference Index

Cloud account patterns pair sandbox compute with sandboxed cloud provider accounts. Each provider has different auth mechanisms, CLI tools, IAM models, and resource lifecycle.

## Container + Cloud

- `container-plus-aws.md` — cloud-client container paired with a sandboxed AWS account
- `container-plus-azure.md` — cloud-client container paired with a sandboxed Azure subscription
- `container-plus-gcp.md` — cloud-client container paired with a sandboxed GCP project

## VM + Cloud

- `vm-plus-aws.md` — Full VM workstation paired with a sandboxed AWS account
- `vm-plus-azure.md` — Full VM workstation paired with a sandboxed Azure subscription
- `vm-plus-gcp.md` — Full VM workstation paired with a sandboxed GCP project

## Composite

- `hybrid-plus-cloud.md` — Container + VM + cloud account (hybrid compute with cloud)
- `multi-cloud.md` — Multiple cloud providers in a single track
